# Instructions

When Microsoft calls you, its this script, just press the '#' key in your phone

## Configuration

IMPORTANT: replace `<your-username>` with the name of your actual username  
Windows configuration is found in: "C:\\Users\\`<your-username>`\\AppData\\Local\\roeyba\\doh1-autofill\\config.json"  
Linux configuration is found in: "/home/`<your-username>`/.config/doh1-autofill/config.json"

Configuration template:

```json
{
  "id_number": "<your_id_number>",
  "microsoft_mail": "<your_id_number>@idf.il",
  "microsoft_password": "<your_password>"
}
```
